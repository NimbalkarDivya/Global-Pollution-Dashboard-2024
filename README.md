The **Global Pollution Dashboard 2024** is a data visualization project that analyzes **global air pollution and environmental risk** across countries and continents.  
Instead of only showing pollution levels, the dashboard converts **PM2.5 values into AQI**, making environmental and health impacts easier to understand.

The project highlights pollution severity, air quality categories, development status, and environmental infrastructure indicators such as sanitation, drinking water, and waste management.
